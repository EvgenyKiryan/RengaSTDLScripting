Что дальше?
-----------

Поздравляем! Вы прошли обучающий проект и поняли как создавать новые шаблоны стилей инженерного оборудования в Renga.

* Исходный код примера и скомплированную шаблонную категорию вы можете скачать с Github.
* Присоединяйтесь к сообществу разработчиков категорий оборудования для Renga в Telegram.
* Делитесь своими идеями с разработчиками Renga в службе техподдержки ServiceDesk.
* Пишите на info@rengabim.com для обсуждения сотрудничества с Renga Software по вопросам создания новых категорий оборудования.

Делаем BIM доступным! Happy scripting!
